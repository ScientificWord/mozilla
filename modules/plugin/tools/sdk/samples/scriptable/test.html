XPConnect Scriptable Sample Plug-in

This page contains a test case which demonstrates the work of scriptable plug-in with Mozilla. The example plug-in occupies the area right below this text, and you should see a frame the plug-in draws around its window. Below the plug-in window there are two buttons. Clicking on the buttons will result in calling native plugin methods from JavaScript. Show Version will instruct the plug-in to retrieve the Mozilla user agent string and display it in the plug-in window, Clear button will call plug-in method to erase the window.
