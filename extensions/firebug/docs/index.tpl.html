Firebug Releases

The latest stable release is available on the front page.

Firebug 1.2 Betas

Firebug 1.2 focuses on performance improvements and support for Firefox 3. The 1.2 features are complete and we are actively fixing bugs. Please read the 1.2 release notes before installing. Recommended for Firefox 3.

Firebug 1.2X is the same code as Firebug 1.2 but with internal debug tracing available for Firebug extension developers.

Firebug 1.1 Betas

Firebug 1.1.0 has had many beta tests and is the version recommended for Firefox 2

Firebug 1.0.5

Firebug 1.0.5 (aka 1.05) was released in May 2007. This version has been widely used. Before reporting bugs, please try Firebug 1.2

ChromeBug Alphas

ChromeBug is Firebug for Firefox extensions.

  • Requires Firebug 1.2.
  • Works with Chromelist.
  • Works on Firefox 3.0 and later
  • To run chromebug: firefox.exe -chrome chrome://chromebug/content/chromebug.xul -firefox
  • Currently unstable, testing install and bugs in FF3, and comments welcome.
