Init Steps: Come to this page with caching off.
Verify: Make sure the selects are initialized properly. The only one with multiple values in it should be the bottom right.
Reset Steps: Change some values in every box (deselect at least one, select at least one) and hit Reset.
Verify: The values should be the same as they were at Init.
Restore Steps: Change some values in every box (as above), go to Google, and hit Back.
Verify: The values should stay at how you changed them.
No Restore Steps: Change some values in every box (as above), and hit shift+reload.
Verify: The values should be reset.
