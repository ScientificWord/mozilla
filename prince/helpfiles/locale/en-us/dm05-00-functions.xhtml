Function Definitions

Definitions in Scientific WorkPlace and Scientific Notebook enable you to define a symbol to be a mathematical object, and to define a function using an expression or a collection of expressions. Function definition is a powerful tool. Before elaborating on definitions, we discuss criteria for names of functions, constants, and expressions.
