Information for Power Users Setting program optionsSetting Program Options Use Tools → Preferences to see and set certain options relevant to saving and opening documents and much more (E.g. change default startup shell, whether to automatically save, etc.). On the Macintosh, Macintosh Users Note this command is available through the Application menu that appears in bold to the right of the Apple menu: SWP for Scientific WorkPlace, SW for Scientific Word, or SNB for Scientific Notebook.
