Storing Data in Lists and Matrices

You can store data in lists or in matrices. Numbers in a list should be separated by commas, with the numbers and commas both in mathematics mode. Lists can be plain or enclosed in brackets. A list of data is also referred to as a set of data. A list can be reshaped into a matrix. To reshape a comma-delimited list or set of data into a matrix If the list is in text, select the list and choose Insert → Math to change it to mathematics. Place the insert point in the list or set and choose Compute → Matrices → Reshape. In the dialog box that appears, type a number for Columns and choose OK. Select the matrix with the mouse, choose Insert → Math objects → Brackets and select appropriate brackets.
