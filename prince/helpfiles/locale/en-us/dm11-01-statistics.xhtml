Storing Data in Lists and Matrices

You can store data in lists or in matrices. Numbers in a list should be separated by commas, with the numbers and commas both in mathematics mode. Lists can be plain or enclosed in brackets. A list of data is also referred to as a set of data.
In this section: Lists and Matrices Importing Data from an ASCII File
Lists and Matrices A list can be reshaped into a matrix. To reshape a comma-delimited list or set of data into a matrix If the list is in text, select the list and choose Insert > Math to change it to mathematics. Place the insert point in the list or set and choose Compute > Matrices > Reshape. In the dialog box that appears, type a number for Columns and choose OK. Select the matrix with the mouse, choose Insert > Math Objects > Brackets and select appropriate brackets. One rather general method for converting data from a text file to a mathematical list or matrix is outlined below. Look under Help for additional options for importing data. If the data is a list of numbers separated by commas, select the data with click and drag, then choose Insert > Math to change it to mathematics. Then use the data as a list, or use the techniques described in the previous section to convert the data to a vector or matrix. If the data is stored as a column of numbers, it will come in as inline numbers separated by spaces. To change numbers separated by spaces to a list Select the data with click and drag. Choose Edit > Find and Replace. In the box for Find text, type a normal space. In the box for Replace with, type a comma followed by an Allowbreak (choose Insert > Breaks, check Allowbreak, choose OK). Choose Replace All. Select the data with click and drag and choose Insert > Math.
