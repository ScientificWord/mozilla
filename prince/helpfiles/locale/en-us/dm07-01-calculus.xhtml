Calculus Expressions To calculate a derivative or an integral Enter a derivative or integral in standard mathematical notation. Choose Compute → Evaluate. [ Return to beginning of section ] Derivatives To calculate the derivative ⅆ ⅆ x x ⁢ sin ⁡ x Choose Insert → Math objects → Fraction Type d in the numerator, and press tab to take the insert point to the denominator. Type ⅆ x and press spacebar to put the insert point back in line, then type x ⁢ sin ⁡ x . Leave the insert point in the expression ⅆ ⅆ x x ⁢ sin ⁡ x and choose Compute → Evaluate. Compute → Evaluate ⅆ ⅆ x x ⁢ sin ⁡ x = sin ⁡ x + x ⁢ cos ⁡ x
Reminder
The mathematics shown above depicts both what you enter, ⅆ ⅆ x x ⁢ sin ⁡ x , and the result of the command Compute → Evaluate, = sin ⁡ x + x ⁢ cos ⁡ x .
[ Return to beginning of section ] Integrals To calculate the definite integral ∫ 0 π x ⁢ sin ⁡ x ⁢ ⅆ x Choose Insert → Math objects → Operator and select ∫ . Choose Insert → Math objects → Subscript and type the lower limit 0 in the subscript box. Press tab and enter the upper limit π in the superscript box. Press spacebar to put the insert point back in line, and type x ⁢ sin ⁡ x ⁢ ⅆ x . Leave the insert point in the expression ∫ 0 π x ⁢ sin ⁡ x ⁢ ⅆ x and choose Compute → Evaluate. Compute → Evaluate ∫ 0 π x ⁢ sin ⁡ x ⁢ ⅆ x = π
Symbol Toolbar
Select π from the Symbol toolbar. To display the Symbol toolbar, choose View → Toolbars → Symbol toolbar.
Integral and area
Compare the area under the curve y = x ⁢ sin ⁡ x between 0 and π and the area of the rectangle having sides of length 1 and π .
