Creating Animated 2D Plots

Animated 2D Rectangular Plots To edit the animation interval Open the Graph User Settings dialog box, Items Plotted page and click Edit Plot. Open the animation page and type the Start time (seconds), End time (seconds), and Frames per second. The following animated plot shows the graph of y = sin ⁡ 2 ⁢ x smoothly transform itself into the graph of y = cos ⁡ 3 ⁢ x .



 Compute > Plot 2D Animated > Rectangular ( 1 − t ) ⁢ sin ⁡ 2 ⁢ x + t ⁢ cos ⁡ 3 ⁢ x Interval: 0 < t < 1 , − 6 < x < 6 When you click this plot, the plotting tools become visible near the top of the screen. Move the slider to control the animation or click the triangle to run an animation.
