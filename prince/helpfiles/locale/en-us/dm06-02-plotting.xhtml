Activating the Interactive Tools for 2D Plots When a plot frame is selected, several plotting tools become visible just below other toolbars at the top of the screen.
 Hover over a tool with your mouse for a tool tips description. To display plotting tools Click the plot. [ Return to beginning of section ] Reset Viewpoint Click the Reset Viewpoint tool to reset the viewpoint to a default position. [ Return to beginning of section ] Fit to Contents Click the Fit to contents tool to fit the plot in the frame. [ Return to beginning of section ] Select Click the Select tool to select the plot inside the frame. [ Return to beginning of section ] Translate the View Changing the Plot Intervals reveals different portions of a plot. To see different portions of a plot in an interactive way, you can translate the view with the Moving tool . The Moving tool shows four arrows, similar to the north, south, east, west arrows commonly shown on a map. To translate the view Select the Moving tool. Move the mouse pointer over the view. Click and drag in the direction you want to translate the plot. As you click and drag, a moving rectangle will indicate where the translated plot will appear when the mouse button is released. [ Return to beginning of section ] Plot Coordinates The default tool for 2D plots is the Query tool . This is used to view the coordinates of points on a plot. To view plot coordinates Select the Query tool. Move the mouse pointer to a point of interest on the plot. Press and hold the left mouse button to view of coordinates. [ Return to beginning of section ] Zoom In and Out The Zoom in tool shows a magnifying glass with a plus and the Zoom out tool a magnifying glass with a minus. To zoom in or out Click the Zoom in or Zoom out tool. Move the mouse pointer over the view. Click the plot to zoom in or zoom out, depending on the selected tool. [ Return to beginning of section ] Snapshot Click the Snapshot tool to create a graphics image snapshot of the plot. To create a snapshot Select the Snapshot tool. The Save shapshot dialog appears. Use the directory pane to navigate to a different location if you wish to save the file elsewhere from the default location shown. You can also choose to change the default file name. Be sure to use a valid name and avoid characters that may be reserved by your operating system, such as * and /.) Click Save. The plot shapshot graphics file is created.
