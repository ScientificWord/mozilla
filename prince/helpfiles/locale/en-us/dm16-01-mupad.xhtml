Constants in Scientific WorkPlace, Scientific Notebook, and MuPAD

Common constants can be expressed in ordinary mathematical notation. See Customizing the Program for Computing for some choices for the appearance of some of the constants.
SWP/SNB MuPAD Comments
e exp(1) or E base of natural logs
i or j (see Complex Numbers) I imaginary unit: − 1
π PI circular constant
gamma EULER lim n → ∞ ( ∑ m = 1 n 1 m − ln ⁡ n )
∞ infinity positive real infinity
true TRUE Boolean true
false FALSE Boolean false
FAIL or undecidable FAIL, UNKNOWN, Either the answer cannot be determined
or UNDEFINED or a nonexistent function was used.
