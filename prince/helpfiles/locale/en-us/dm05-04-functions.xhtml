Assumptions About Variables

In some situations it is useful to restrict the domain of a variable. For example, you may want the variable to assume only positive values or only real values. Such restrictions are made with the function assume .

Assumptions and Restraint Functions The functions available for making or checking or removing assumptions are assume    additionally    about    unassume These functions place restraints on specific variables or on all variables, provide information on the restraints, or remove restraints. The function assume enables you to place a restraint on a variable. The function additionally allows you to place additional restraints. The function about returns information on the restraints. The function unassume removes restraints. Allowable assumptions include real   complex   integer   positive   negative   nonzero The following assumptions are also allowed for real variables x and y , and complex variable z : x < y    x < 3    x ≠ 0    x ≤ y    x ≥ 5 Im ⁡ ( z ) > 0    Re ⁡ ( z ) < 0    Re ⁡ ( z ) ≠ 0 To enter the names of the restraint functions and assumptions Choose Insert > Math Objects > Math Name and type the name in the input box or select from the list of names.
