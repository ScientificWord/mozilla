Systems of Differential Equations

Systems consisting of more than one differential equation are handled in a consistent manner. Such problems include initial-value problems and systems of differential equations.
In this section: Exact Solutions Series Solutions
Exact Solutions The statement of some problems requires more than one equation. You enter systems with initial conditions, systems of differential equations, boundary-value problems, or a mixture of these problems using n × 1 matrices, where n is the number of equations and conditions involved. To create a system of differential equations in a matrix Choose Insert > Math Objects > Matrix. Select 1 column, set the number of rows equal to the number of equations, and choose OK. Choose View and select Helper Lines and Input Boxes to show where to enter the required equations. Type the equations, one to a row. To create a system of differential equations in a display Choose Insert > Math Objects > Display. Choose View and select Helper Lines and Input Boxes to show where to type the required equations. Type the equations, one to a row, pressing Enter to create each new row as needed. [ Return to beginning of section ] To solve a system of differential equations Leave the insert point in the matrix or display. Choose Compute > Solve ODE > Exact, or Choose Compute > Solve ODE > Laplace. Series Solutions

The following examples illustrate series solutions to two types of differential equations with boundary conditions. To solve a differential equation by the series method Enter the equation and boundary conditions in a display or one-column matrix. Place the insert point in the display or matrix and choose Compute > Solve ODE > Series. If the Math Computation Arguments dialog appears, type a variable, center, and order in the dialog box. For the following examples, the series center is 0 and order is 5 (the default values).
