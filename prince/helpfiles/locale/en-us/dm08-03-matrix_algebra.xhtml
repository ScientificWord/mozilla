Row Operations and Echelon Forms

One of the elementary applications of matrix arrays is storing and manipulating coefficients of systems of linear equations. The various steps that you carry out in applying the technique of elimination to a system of linear equations a 1 ⁣ 1 ⁢ x 1 + a 1 ⁣ 2 ⁢ x 2 + … + a 1 ⁣ n ⁢ x n = b 1 a 2 ⁣ 1 ⁢ x 1 + a 2 ⁣ 2 ⁢ x 2 + … + a 2 ⁣ n ⁢ x n = b 2 ⋮        ⋮ ⋮ a m ⁣ 1 ⁢ x 1 + a m ⁣ 2 ⁢ x 2 + … + a m ⁣ n ⁢ x n = b m can be applied equally well to the matrix of coefficients: [ a 1 ⁣ 1 a 1 ⁣ 2 … a 1 ⁣ n b 1 a 2 ⁣ 1 a 2 ⁣ 2 … a 2 ⁣ n b 2 ⋮ ⋮ … ⋮ ⋮ a m ⁣ 1 a m ⁣ 2 … a m ⁣ n b m ] For this and numerous other reasons, you perform elementary row operations on matrices. The goal of elementary row operations is to put the matrix in a special form, such as a row echelon form, in which the number of leading zeroes increases as the row number increases.

[ Return to beginning of section ] Gaussian Elimination and Row Echelon Form The Matrices menu provides the choices Fraction-free Gaussian Elimination, Gaussian Elimination, and Reduced Row Echelon Form for obtaining a row echelon form. The three row echelon forms that can be obtained from the Matrices submenu are illustrated in the following examples. Compute > Matrices > Fraction-free Gaussian Elimination [ a b c d ] , fraction-free Gaussian elimination: [ a b 0 a ⁢ d − b ⁢ c ] [ 8 2 3 2 − 5 8 ] , fraction-free Gaussian elimination: [ 8 2 3 0 − 44 58 ] Compute > Matrices > Gaussian Elimination [ a b c d ] , Gaussian elimination: [ a b 0 1 a ⁢ ( a ⁢ d − b ⁢ c ) ] [ 8 2 3 2 − 5 8 ] , Gaussian elimination: [ 8 2 3 0 − 11 2 29 4 ] Compute > Matrices > Reduced Row Echelon Form [ a b c d ] , row echelon form: [ 1 0 0 1 ] [ 8 2 3 2 − 5 8 ] , row echelon form: [ 1 0 31 44 0 1 − 29 22 ] The last of these, the reduced row echelon form, produces a matrix satisfying the following conditions: The number of leading zeroes increases as the row number increases. The first nonzero entry in each nonzero row is equal to 1 . Each column that contains the leading nonzero entry for any row contains only zeroes above and below that entry.
