Formulas

The Formula dialog box provides a way to enter an expression and a Compute operation. What appears in the document window is the result of the operation and depends upon active definitions of variables that appear in the expression you have entered in the formula. Formulas remain active in your document---that is, changing definitions of relevant variables changes the data on the screen as well as in printed documents. Formulas are useful for writing multiple choice examinations with variations.

Creating a Formula Object To insert a formula Choose Insert > Math Objects > Formula. Type an expression in the input box. In the Operation area, select the operation you want to perform on the expression. (Click the arrow at the right of the box for a list of available operations.) Choose OK. The results of the operation will be displayed in your document window and in print previews. Under Tag Properties, choose Special Objects. Under Special Objects choose Formula and click Modify. Select background color and choose OK. The next example outlines a way for constructing them manually. The questions in an examination depend on definitions that are made globally for each document---they're not local to each question or variant. This means that you often should use a Math Name instead of a single character name for variables. A sample question is shown in the next example. The variables a1 and b1 shown in the next example are math names. Using a Math Name instead of a single character name for each variable avoids conflicts with other definitions in the document. To enter a Math Name in a Formula box Choose Insert > Math Objects > Formula. With the insert point in the Formula box, choose Insert > Math Objects > Math Name. Type or select a Math Name and choose OK. Example You can create an examination with variations by making different definitions for the variables such as the a1 and b1 shown in the following question. Turn on Helper Lines in your document and look for background color to check that all appropriate entries are formulas. For which values of the variable x is a1 x − b1 < 0 ? x < b1 / a1 x > b1 / a1 x > b1 x < b1 None of these Example Define a1 = 2 and b1 = 5 by placing the insert point in each equation and choosing Compute > Definitions > New Definition. In your document window you should see For which values of the variable x is 2 ⁢ x − 5 < 0 ? x < 5 / 2 x > 5 / 2 x > 5 x < 5 None of these After printing a quiz, make different definitions for all the variables such as a1 and b1 to obtain variations of the quiz.
