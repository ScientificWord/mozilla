Absolute value The absolute value of a number z , the distance of z from zero, is denoted | z | . To put vertical bars around an expression Select the expression. Choose Insert → Math objects → Brackets. Click the vertical bracket and click OK. Caution The vertical lines from the symbol panel of Binary Relations will not be interpreted as absolute-value symbols in computations. Although they appear similar, they are not the same symbols. The keyboard vertical line will work to build an absolute value, but expanding brackets are less vulnerable to misinterpretation. Type Ctrl+\ for expanding absolute value bars. To compute an absolute value Place the insert point in an expression between vertical bars. Choose Compute → Evaluate. Leave the insert point in the expression and choose Evaluate. Compute → Evaluate max ( 13 2 , − 63 , 7.3 ) = 7.3 min ( 13 2 , − 63 , 7.3 ) = − 3 ⁢ 7 To enter function names for maximum and minimum Choose Insert → Math objects → Math name. Select from the list. Or Choose Insert → Math. Type max or min . The binary operations join ∨ and meet ∧ also give maximum and minimum. Compute → Evaluate 27 ∨ 65 2 ∨ − 14 = 65 2 27 ∧ 65 2 ∧ − 14 = − 14 To enter vee and wedge symbols for maximum and minimum Click Binary Operations ± + ▾ on the Symbol toolbar. Select ∨ or ∧ from the panel. Tip To make the symbol toolbar visible, choose View → Toolbars and check Symbol toolbar. To find the maximum or minimum of a finite sequence, enter the limits on the integer variable as a subscript on max or min , either in the form of a double inequality such as 1 ≤ n ≤ 10 or as membership in an interval such as k ∈ [ 1 , 10 ] . Compute → Evaluate max 1 ≤ n ≤ 10 ( sin ⁡ n ) = sin ⁡ 8 max 1 ≤ n ≤ 10 ( sin ⁡ 1.5 ⁢ n ) = 0. 997 49 min k ∈ [ 1 , 10 ] ( cos ⁡ k ) = cos ⁡ 3 min k ∈ [ 1 , 10 ] ( cos ⁡ 2.6 ⁢ k ) = − 0. 994 18 Compute → Evaluate numeric max 1 ≤ n ≤ 10 ( sin ⁡ n ) ≈ 0.989 36 max − 2 ≤ x ≤ 2 ( x 3 − 6 ⁢ x + 3 ) ≈ 8.0 min k ∈ [ 1 , 10 ] ( cos ⁡ k ) ≈ − 0. 989 99 min k ∈ [ 1 , 10 ] ( cos ⁡ 2.6 ⁢ k ) ≈ − 0. 994 1 Note that the functions max and min look only at the sequence of values for integer variables. The notations x ∈ [ − 2 , 2 ] and − 2 ≤ x ≤ 2 both indicate that x assumes the range of values in the 5-element set { − 2 , − 1 , 0 , 1 , 2 } . For example, with max − 2 ≤ x ≤ 2 ( x 3 − 6 ⁢ x + 3 ) , the maximum is picked from among values of x 3 − 6 ⁢ x + 3 for x = − 2 , − 1 , 0 , 1 , 2 . In particular, the expression max − 2 ≤ x ≤ 2 ( x 3 − 6 ⁢ x + 3 ) is not the maximum of the continuous polynomial function x 3 − 6 ⁢ x + 3 on the interval − 2 ≤ x ≤ 2 . To find a greatest integer value Place the insert point in an expression between floor brackets. Choose Compute → Evaluate. To find a smallest integer value Place the insert point in a number between ceiling brackets. Choose Compute → Evaluate. [ Return to beginning of section ] Checking Equality and Inequality You can verify equalities and inequalities with the command Check Equality or with the function istrue . There are three possible responses: true, false, and undecidable. The latter means that the test is inconclusive and the equality may be either true or false. The computational engine may use probabilistic methods to check equality, and there is a very small probability that an equation judged as true is actually false. Some expressions cannot be compared by this method—hence the inconclusive response. Use Check Equality to check equalities and inequalities To check whether an equality is true or false Place the insert point in the equation. Choose Check Equality. Compute → Check equality e i ⁢ π = − 1 is TRUE π = 3.14 is FALSE arcsin ⁡ sin ⁡ x = x is FALSE

 You can also use Check Equality to check an inequality between two numbers. Set the difference of the two numbers equal to the absolute value of the difference, place the insert point in the equation, and choose Check Equality.

 Compute → Check equality 9 8 − 8 9 = | 9 8 − 8 9 | is TRUE π e − e π = | π e − e π | is FALSE

 These results verify that 9 8 − 8 9 ≥ 0 , or 9 8 ≥ 8 9 ; and that π e − e π < 0 , or π e < e π . Use istrue to check equalities and inequalities Type i ⁢ s ⁢ t ⁢ r ⁢ u ⁢ e in mathematics mode to get the function name istrue , or create it as a Math name in the Insert → Math objects → Math name dialog box. Evaluate this function at an equation or inequality to test it.

 Compute → Evaluate istrue ⁡ ( 9 8 < 8 9 ) = FALSE istrue ⁡ ( 2 + 2 = 4 ) = TRUE istrue ⁡ ( π e < e π ) = TRUE istrue ⁡ ( ( 2 ) 2 = 2 ) = TRUE

 Use logical operators to check equalities and inequalities The operators ∧ (AND) and ∨ (OR) can be used as logical operators. The statement α ∧ β is true if and only if both α and β are true. The statement α ∨ β is true if and only if at least one of α and β is true. Using a tautology such as 0 = 0 or 1 = 1 as one of the statements, you can test the truth or falsity of another equation or inequality.

 Compute → Evaluate ( 5 6 < 6 5 ) ∧ ( 1 = 1 ) = FALSE ( 5 6 > 6 5 ) ∧ ( 1 = 1 ) = TRUE ( 5 6 > 6 5 ) ∨ ( 1 = 1 ) = TRUE ( 5 6 < 6 5 ) ∨ ( 1 = 1 ) = TRUE ( 1 = 1 ) ∨ ( 1 = 0 ) = TRUE ( e π = π e ) ∧ ( 0 = 0 ) = FALSE

 Evaluate numerically to check inequalities In some cases, you can recognize an inequality by inspection after applying Evaluate numeric to each of the numbers.

 Compute → Evaluate numeric 9 8 ≈ 1.125 and 8 9 ≈ 0.88889 π e ≈ 22.459 and e π ≈ 23.141

From this we see that 9 8 > 8 9 and π e < e π . [ Return to beginning of section ] Union, Intersection, and Difference Numbers or expressions to be used for computing should be entered in mathematics mode and appear red on your screen. If that is not the case, choose Insert → Math to change the expression to mathematics. Following are some of the basic built-in relations (union, intersection, and difference of sets). Union You can find the union of two or more finite sets with Evaluate, by using the symbol ∪ between the sets. Compute → Evaluate { 1 , 2 , 3 } ∪ { a , b , c } = { 1 , 2 , 3 , a , b , c } { 1 , 2 , 3 } ∪ ( { 3 , 5 } ∪ { 7 } ) = { 1 , 2 , 3 , 5 , 7 } { 2 , π , 3.9 , r } ∪ { a , b , c } = { 3. 9 , π , a , b , c , r , 2 } Intersection You can find the intersection of two or more finite sets with Evaluate, using the symbol ∩ between the sets. Compute → Evaluate { 1 , 2 , 3 } ∩ { 2 , 4 , 6 } = { 2 } { a , b , c , d } ∩ { d , e , f } = { d } { 1 , 2 , 3 } ∩ { a , b , c } = ∅ { 1 , 2 , 3 } ∩ ∅ = ∅ If two sets have no elements in common, their intersection is the empty set, denoted by empty brackets { ​ } or the symbol ∅ . To enter the symbol ∅ for the empty set, select it from the Miscellaneous Symbols panel under ∞ ∂ . Difference You can find the difference of two finite sets with Evaluate, by placing between the sets a backslash \ or the setminus symbol ∖ from the Binary Operations panel. Compute → Evaluate { 1 , 2 , 3 , 4 } \ { 2 , 4 } = { 1 , 3 } { a , b , c , d } \ { d , e , f } = { a , b , c } { 1 , 2 , 3 } ∖ { a , b , c } = { 1 , 2 , 3 } { 1 , 2 , 3 } ∖ { 1 , 2 , 3 } = ∅ Combinations of union, intersection, and difference You can evaluate combinations of union, intersection, and difference after grouping expressions appropriately with expanding parentheses.
