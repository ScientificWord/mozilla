Matrices Modulo m You can reduce the entries of an integer matrix modulo m . To reduce a matrix A modulo m Type the expression A mod m . Choose Compute > Evaluate. Compute > Evaluate [ 5 8 9 4 ] mod 3 = [ 2 2 0 1 ] ( 3 7 5 5 4 8 2 0 5 ) − 1 mod 11 = ( 9 9 3 2 5 1 3 3 10 ) ( 3 7 5 5 4 8 2 0 5 ) ⁢ ( 9 9 3 2 5 1 3 3 10 ) mod 11 = ( 1 0 0 0 1 0 0 0 1 ) [ Return to beginning of section ]
Example A 2 × 2 block cipher is given by [ y 1 y 2 ] = [ a 1 ⁣ 1 a 1 ⁣ 2 a 2 ⁣ 1 a 2 ⁣ 2 ] ⁢ [ x 1 x 2 ] mod 26 where the x i 's represent plaintext, the y i 's represent ciphertext, and the matrix entries are integers. For example, [ 5 8 2 7 ] ⁢ [ 4 11 ] mod 26 = [ 4 7 ] means that the plaintext pair [ E , L ] (two adjacent letters in the secret message "Elroy was here") gets mapped to the ciphertext pair [ E , H ] , using the correspondence A ↔ 0 , B ↔ 1 , C ↔ 2 , … , Z ↔ 25 . Given the ciphertext, you can recover the plaintext by computing the inverse of the two-by-two matrix modulo 26 . For example, [ 5 8 2 7 ] − 1 mod 26 = [ 25 16 4 3 ] and hence [ 25 16 4 3 ] ⁢ [ 4 7 ] mod 26 = [ 4 11 ] recovers the original plaintext. You can handle longer messages by replacing the column vector [ E L ] by the matrix [ E R Y A H R L O W S E E ] and calculating one matrix product [ 25 16 4 3 ] ⁢ [ 4 17 24 0 7 17 11 14 22 18 4 4 ] mod 26 = [ 16 25 16 2 5 21 23 6 6 2 14 2 ] .
[ Return to beginning of section ] Polynomials Modulo m The mod function can also be combined with polynomials to reduce each of the coefficients modulo m . To reduce integer polynomial coefficients modulo m Type mod m immediately to the right of the polynomial and choose Compute > Evaluate. Compute > Evaluate x 5 + 9 ⁢ x 4 − x 3 + 7 ⁢ x − 2 mod 5 = x 5 + 4 ⁢ x 4 + 4 ⁢ x 3 + 2 ⁢ x + 3 Given a prime p , the set of polynomials with coefficients reduced modulo p is a ring, denoted by G ⁢ F p ⁢ [ x ] . To calculate the product of polynomials a ⁡ ( x ) and b ⁡ ( x ) in G ⁢ F p ⁢ [ x ] Expand the product a ⁡ ( x ) ⁢ b ⁡ ( x ) . Reduce the product modulo p . To calculate the product of 4 ⁢ x 5 + 5 ⁢ x + 3 and 6 ⁢ x 4 + x 3 + 3 in G ⁢ F 7 ⁢ [ x ] , do the following two operations.

 Compute > Expand ( 4 ⁢ x 5 + 5 ⁢ x + 3 ) ⁢ ( 6 ⁢ x 4 + x 3 + 3 ) = 24 ⁢ x 9 + 4 ⁢ x 8 + 42 ⁢ x 5 + 23 ⁢ x 4 + 3 ⁢ x 3 + 15 ⁢ x + 9 Compute > Evaluate 24 ⁢ x 9 + 4 ⁢ x 8 + 42 ⁢ x 5 + 23 ⁢ x 4 + 3 ⁢ x 3 + 15 ⁢ x + 9 mod 7 = 3 ⁢ x 9 + 4 ⁢ x 8 + 2 ⁢ x 4 + 3 ⁢ x 3 + x + 2 The sum of 4 ⁢ x 5 + 5 ⁢ x + 3 and 6 ⁢ x 4 + x 3 + 3 in G ⁢ F 7 ⁢ [ x ] is slightly simpler. Compute > Evaluate ( 4 ⁢ x 5 + 5 ⁢ x + 3 ) + ( 6 ⁢ x 4 + x 3 + 3 ) mod 7 = 4 ⁢ x 5 + 6 ⁢ x 4 + x 3 + 5 ⁢ x + 6 [ Return to beginning of section ] To factor a polynomial a ⁡ ( x ) in G ⁢ F p ⁢ [ x ] Type the expression a ⁡ ( x ) mod p . Choose Compute > Factor. To factor x 16 + x in G ⁢ F 2 ⁢ [ x ] , apply the command Factor to the expression x 16 + x mod 2 .

 Compute > Factor x 16 + x mod 2 = x ⁢ ( x + 1 ) ⁢ ( x 2 + x + 1 ) ⁢ ( x 4 + x + 1 ) ⁢ ( x 4 + x 3 + 1 ) ⁢ ( x 4 + x 3 + x 2 + x + 1 ) The polynomial x 16 + x factors as the product of all of the irreducible polynomials of degrees 1 , 2 , and 4 . In particular, x 2 + x + 1 is the only irreducible polynomial of degree 2 in G ⁢ F 2 ⁢ [ x ] . [ Return to beginning of section ]
