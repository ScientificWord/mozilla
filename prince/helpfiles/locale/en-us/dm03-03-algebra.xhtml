Substituting for a Variable To substitute a number or new expression for a variable Enclose the expression in square brackets. Choose Insert > Math Objects > Subscript. Type an assignment for the variable in the subscript input box. Choose Compute > Evaluate. Compute > Evaluate [ x 2 + 2 ⁢ x − 3 ] x = a = a 2 + 2 ⁢ a − 3 [ x 2 + 2 ⁢ x − 3 ] x = 5 = 32 [ x + y ] x = y + z = 2 ⁢ y + z [ x + y ] y = x − z = 2 ⁢ x − z [ x 2 + 2 ⁢ x − 3 ] x = y − z = 2 ⁢ y − 2 ⁢ z + ( y − z ) 2 − 3
Note
The expression in the subscript is an assignment for the variable on the left of the equals sign. Notice that, in particular, x = a and a = x are not equivalent assignments; and x = y + z , y = x − z , and z = x − y are not equivalent assignments.
