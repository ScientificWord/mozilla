Creating and Editing Matrices

You can create a matrix via the Matrix dialog box or from the Matrices menu with Fill Matrix or Random Matrix. There is also a keyboard shortcut.
In this section: Creating Matrices with the Matrix Dialog Box Creating Matrices with Fill Matrix Creating Matrices with Random Matrix Adding Matrix Delimiters Revising Matrices Concatenating and Stacking Matrices Reshaping Lists and Matrices
Several methods for creating matrices are described in the ensuing sections. Each method involves different choices, as described in the following paragraphs. Creating Matrices with the Matrix Dialog Box To create a matrix Choose Insert > Math Objects > Matrix. Select the number of rows and columns by clicking in the lowest, right-most box you want. Choose OK. Type or copy entries into the input boxes. The entries can be any valid mathematical expression. Both real and complex numbers are legitimate entries, as well as algebraic expressions. See other options under Fill Matrix and Random Matrix. [ Return to beginning of section ] Matrix Delimiters You can make choices in the View menu that affect the appearance of matrices on the screen. Helper Lines and Input Boxes can be shown or hidden. The default is to show them to make it easier to handle entries on the screen. Matrix helper lines and input boxes normally do not appear when you preview or print the document. It is standard to enclose a matrix in brackets, either built-in or added manually. These two options provide the same screen appearance and mathematical properties. They differ only under Typeset---in which case, the built-in brackets fit more tightly around the matrix entries than added brackets. If you have a matrix without built-in delimiters, you will generally want to add brackets around it. The result of an operation on matrices usually appears with the same brackets as the original matrices. To add brackets to a matrix Select the matrix with the mouse. Choose Insert > Math Objects > Brackets. Select the desired brackets and choose OK. To set the default matrix delimiters Choose Tools > Preferences > Computation > Matrices. Select the desired option: none, [ ], or ( ). [ Return to beginning of section ] Revising Matrices You can add or delete rows, columns, or a full block of rows or columns from a matrix. The alignment of rows and columns can be reset. Entries in a rectangular block can be deleted or replaced. Adding Rows and Columns To add rows or columns to a matrix Select the matrix by placing the insert point in a cell of the matrix or by placing the insert point at the right of the matrix (but not outside of any brackets). Choose Edit > Insert Matrix Rows, or Edit > Insert Matrix Columns. Make appropriate choices from the dialog that appears and choose OK. Deleting Rows and Columns To delete a block of rows or columns Select a block of rows or columns with the mouse or with Shift+arrow. Press Del. You can also use the procedure described above to delete entries from a rectangular block that does not include a complete row or column of a matrix. The choices Insert Row(s) and Insert Column(s) appear on the Edit menu only when a matrix is selected. If they do not appear, reposition the insertion point or select the matrix with click and drag, being careful to select only the inside of the matrix---that is, not including the exterior Helper Lines. To lengthen a vector represented as an n × 1 or 1 × n matrix Place the insert point in the last input box and press Enter. To shorten a vector represented as an n × 1 or 1 × n matrix Place the insert point in the last input box and press Backspace. You can start with a display box, or the input boxes that appear with the fraction, radical, or bracket buttons, and make similar changes. [ Return to beginning of section ] Changing Alignment To change the alignment of entries Select the matrix using the mouse or, starting with the insert point at the left of the matrix, press Shift+right arrow. Choose Edit > Properties and make appropriate choices from the dialog box that appears. Replacing a Rectangular Block You can replace a rectangular block in an existing matrix with Copy and Paste or with Fill Matrix. To replace a rectangular block with Copy and Paste Copy a rectangular matrix to the clipboard with Edit > Copy. With the mouse or Shift+arrow, select a rectangular portion of the same dimensions in any matrix and choose Edit > Paste. To change a matrix with Fill Matrix Select a rectangular portion of the matrix with the mouse or Shift+arrow. Choose Compute > Matrices > Fill Matrix. Choose one of the items from the dialog. Choose OK. The selected region of the matrix is filled with the entries that you chose. To change the lower-right 2 × 2 corner of the matrix to the zero matrix Select the lower-right 2 × 2 corner of the matrix using the mouse. Choose Compute > Matrices > Fill Matrix. Choose Zero. To make a matrix from a list Place the insert point within the list. Choose Compute > Matrices > Reshape. Specify the number of columns. The number of rows depends on the length of the list. Extra input boxes at the end are left blank.
