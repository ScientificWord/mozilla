Automatic Substitution

You can create your own names for variables and functions that will be automatically recognized. [ Return to beginning of section ] Example: To recognize b ⁢ o ⁢ b as the variable Robert Choose Tools > Automatic Substitution. Type bob in the Keystrokes box Choose Type of Substitution: Simple substitution. Choose Context for Substitution: In Math. Choose Enable or Disable Auto Substitution: In Math. Leave the insert point in the Substitution entry box. Choose Insert > Math Objects > Math Name. Type Robert in the Name box. Choose Name Type: Variable. Check This is an engine string. Check Add automatic substitution. Choose Apply. (The name `Robert' should appear in the Substitution Box in the Automatic Substitution box.) In the Automatic Substitution dialog, choose Save. Compute > Definitions > New Definition (In mathematics, type b ⁢ o ⁢ b .) Robert = 17 Compute > Evaluate Robert 2 = 289 For additional examples and information about automatic substitution, see Automatic Substitution, choose Help > Search, or consult the manual Creating Documents with Scientific WorkPlace and Scientific Word.
