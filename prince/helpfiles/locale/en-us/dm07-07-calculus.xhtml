Sequences A sequence { a n } n = 1 ∞ is a function whose domain is the set of positive integers. Calculate limits of sequences by selecting an expression such as lim n → ∞ ( 1 + 1 n ) n and choosing Compute > Evaluate, or by defining a n , writing lim n → ∞ a n , and choosing Compute > Evaluate.

 Compute > Evaluate lim n → ∞ ( 1 + 1 n ) n = e The terms of the sequence can be defined as function values, with the subscript as function argument. To define the sequence a n = ( 1 + 1 n ) n With the insert point in the equation a n = ( 1 + 1 n ) n , choose Compute > Definitions > New Definition. In the Interpret Subscript dialog that appears, check A Function Argument and choose OK. Compute > Definitions > New Definition (check A Function Argument) a n = ( 1 + 1 n ) n Compute > Evaluate lim n → ∞ a n = e To compute several terms of a sequence With the insert point in mathematics, type seq . The ratio test states that a series ∑ n = 1 ∞ a n converges absolutely (and therefore converges) if lim n → ∞ | a n + 1 a n | = L < 1 To apply the ratio test to the series ∑ n = 1 ∞ n 2 2 n , define a n = n 2 2 n and compute lim n → ∞ a n + 1 a n to find if it is less than 1 .

 Compute > Definitions > New Definition a n = n 2 2 n Compute > Evaluate lim n → ∞ a n + 1 a n = 1 2 Thus, L = 1 2 , which is less than 1 , so the series converges absolutely. Root test The root test states that a series ∑ n = 1 ∞ a n converges absolutely (and therefore converges) if lim n → ∞ | a n | n = L < 1 To apply the root test to the series ∑ n = 1 ∞ n 2 2 n , define a n = n 2 2 n and compute lim n → ∞ | a n | n to find if it is less than 1 .

 Compute > Definitions > New Definition (Subscript is function argument) a n = n 2 2 n Compute > Evaluate lim n → ∞ | a n | n = 1 2 Thus, L = 1 2 , which is less than 1 , showing that this series converges absolutely. [ Return to beginning of section ] Integral test The integral test states that a series ∑ n = 1 ∞ a n converges absolutely if there exists a positive decreasing function f such that f ⁡ ( n ) = | a n | for each positive integer n and ∫ 1 ∞ f ⁡ ( x ) ⁢ ⅆ x < ∞ To verify convergence of the series ∑ n = 1 ∞ n 2 2 n using the integral test, define f by f ⁡ ( x ) = x 2 2 x , compute ∫ 1 ∞ x 2 2 x ⁢ ⅆ x and determine if it is finite.

 Compute > Definitions > New Definition f ⁡ ( x ) = x 2 2 x Compute > Evaluate, Compute > Evaluate Numeric ∫ 1 ∞ x 2 2 x ⁢ ⅆ x = 1 2 ⁢ ln ⁡ 2 + 1 ln 2 ⁡ 2 + 1 ln 3 ⁡ 2 ≈ 5.805497209 Thus, this integral is finite. (Although for f ⁡ ( x ) = x 2 2 x , it is true that f ⁡ ( 1 ) < f ⁡ ( 2 ) < f ⁡ ( 3 ) , you can verify that f is decreasing for x > 3 . In fact, f ⁢ ′ ⁡ ( x ) = 2 ⁢ x 2 x − x 2 2 x ⁢ ln ⁡ 2 is positive only on the interval 0 < x < 2 ln ⁡ 2 = 2.8854 , so f is decreasing on 3 < x < ∞ . Since convergence of a series depends on the tail end of the series only, it is sufficient that the sequence of terms be eventually decreasing.) [ Return to beginning of section ] Maclaurin Series The Maclaurin series of a function f is the series ∑ n = 0 ∞ f ( n ) ⁡ ( 0 ) n ! ⁢ x n where f ( n ) ⁡ ( 0 ) indicates the n th derivative of f evaluated at 0. It is a power series expanded about x = 0 .

 To expand a function f ⁡ ( x ) in a Maclaurin series Place the insert point in the expression f ⁡ ( x ) . Choose Compute > Power Series. Specify Variable, Center, and Order. Choose OK.

The Taylor series of f expanded about x = a is given by ∑ n = 0 ∞ f ( n ) ⁡ ( a ) n ! ⁢ ( x − a ) n and hence is expanded in powers of x − a and centered about a .

 To expand a function f ⁡ ( x ) in a Taylor series in powers of x − a Place the insert point in the expression f ⁡ ( x ) . Choose Compute > Power Series. Specify Variable x , Center a , and Order. Choose OK.

 To find the Taylor series of ln ⁡ x expanded about x = 1 , choose Compute > Power Series. In the dialog box, type the variable x , type the center 1 , and select the desired order of the approximating polynomial.
