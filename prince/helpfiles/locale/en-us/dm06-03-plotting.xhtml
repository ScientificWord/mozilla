Graph User Settings

From the Graph User Settings dialog box, you can change properties of Items Plotted, Axes, Layout, Labelling, and View.

[ Return to beginning of section ] Opening the Graph User Settings Dialog Box To open the Graph User Settings dialog box Click the gray plot frame to select the plot and choose Compute > Edit Plot. Or Double click the gray plot frame. There is a keyboard shortcut to make the Graph User Settings dialog box open when a plot is created, so that you can customize settings before generating the plot. To open the Graph User Settings dialog box while creating a plot Place the insert point to the right of a mathematics expression or function name. Press Ctrl while choosing the plot command. This brings up the Graph User Settings dialog box with the tabbed pages Items Plotted, Axes, Layout, Labelling, and View. Items plotted From the Items Plotted page of the Graph User Settings dialog box you can edit, add, and delete expressions to be plotted. This page shows the Dimensions (2 or 3), the Plot Type (Rectangular, Polar, Implicit, Inequality, Parametric, Conformal, Gradient, Vector Field, ODE, or Approximate Integral), and whether or not the plot is animated. [ Return to beginning of section ] Editing a plot To edit a plot Open the Graph User Settings dialog box, choose the Items Plotted page, and select an Item Number. Select the Line Style (Solid, Dash, or Dots), Point Marker (Squares, Circles, Crosses, Diamonds, Filled Squares, Filled Circles, Filled Diamonds, Stars, or X Crosses), Line Thickness (Thin, Medium, or Thick), and Color. If the graph has a discontinuity and you want asymptotes to be plotted, uncheck Adjust Plot for Discontinuities. Line style The line style can be solid, dash, or dots.

[ Return to beginning of section ] Axes labels The default labels for the axes are x and y for two-dimensional plots, or x , y , and z for three-dimensional plots. You can edit the default labels. [ Return to beginning of section ] Layout Layout properties include the size of a graphic, its placement within your document, and the print and screen display attributes. The defaults for the layout can be changed on the Layout page of the Graph User Settings dialog box. Resizing the frame All plots have an attribute known as fit to frame. When you resize the frame, the plot is resized with it. You can resize the frame either with the mouse or with the Graph User Settings dialog box.



 To resize the frame with the mouse Click to the right of the plot. Move the mouse to the insert point. (The arrow changes to a hand.) Click when you see a hand. (This will create a resizable frame.) Resize the frame by dragging one of the eight handles.



 When the plot frame is selected, eight handles are visible and you can resize the frame by dragging one of the handles. The corner handles leave the opposite vertex fixed while moving the two sides adjacent to the handle, creating a frame that has edges proportional to the original frame . The edge handles move only the corresponding edge in or out. Either type of change stretches or shrinks a plot in the view, along with the frame. Resizing the frame retains the same domain and view intervals. For example, use one of the side handles to create a tall and narrow frame or use one of the handles on the top or bottom to create a short and fat frame . The examples in the previous paragraph illustrate the use of inline plots, one of the two placement options described in the next section. Use the Graph User Settings dialog box to specify frame dimensions precisely.



 To resize the frame with the Graph User Settings dialog box Select the plot and open the Graph User Settings dialog box. Click the Layout tab. In the boxes for Width and Height, set the sizing options you want. Choose OK. [ Return to beginning of section ] Plot Placement Plots can be placed in line, displayed, or floating. [ Return to beginning of section ] Placement options With Scientific Notebook, there are two choices for frame placement---In Line and Displayed. With Scientific WorkPlace you can also choose Floating. Open the Graph User Settings dialog box to see how a frame is placed in your document. To verify and/or change the placement Select the frame and open the Graph User Settings dialog box. Choose the Layout tab and check your choice for Placement. [ Return to beginning of section ] In-line placement An inline frame behaves like a word in the text, in the sense that the frame is pushed along in the line when you enter additional items to the left of it. To change a plot or graphic to in line Select the frame and open the Graph User Settings dialog box. Choose the Layout tab and change the Placement to In Line. When the placement is In Line, you can move it up or down within the line. To move an inline frame up or down with the mouse Select the frame. Drag the frame up or down. The use of the mathematics display, which treats the frame like mathematics, can lead to unpredictable results when you preview or print your document. If the frame appears red on your screen, you can change the frame to text mode by selecting it with the mouse and clicking the Math/Text button.You can drag the plot frame such that its lower edge is resting on the text baseline like , is centered on the line like , hangs with the upper edge at the text baseline like , or rests anywhere in between. To change a plot or graphic to displayed Select the frame and open the Graph User Settings dialog box. Choose the Layout tab and change the Placement to Displayed. Floating frames Floating placement is a typesetting option, available in Scientific WorkPlace or Scientific Word only. Floating frames containing plots aren't anchored to a precise location in your document. Instead, they are positioned when you typeset the document, according to the options you choose for placement: Here, On a Page of Floats, Top of Page, or Bottom of Page. Floating frames can carry numbers, captions, and keys. The number is created automatically by LaTeX unless you suppress it. If you don't typeset, in the File > Preview screen or on paper, floating frames behave like displayed frames. To change a plot or graphic to floating Select the frame and open the Graph User Settings dialog box. Choose the Layout tab and change the Placement to Floating. [ Return to beginning of section ]
