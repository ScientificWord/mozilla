Creating Animated 2D Plots

Animated 2D Rectangular Plots To create a 2D animation Type an expression in x and t , and choose Compute > Plot 2D Animated > Rectangular. Compute > Plot 2D Animated > Rectangular x ⁢ t When you click this plot, the plotting tools become visible near the top of the screen. Use the slider to control the animation or click the triangle to view the animation.
