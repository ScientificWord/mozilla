Definite Integrals

Informally, the definite integral of f from a to b , written as ∫ a b f ⁡ ( t ) ⁢ ⅆ t , is the net area between the graph of f and the interval [ a , b ] ---that is, the area of the region that is above the interval [ a , b ] and below the graph of f , minus the area of the region that is below the interval [ a , b ] and above the graph of f .

[ Return to beginning of section ] The Definite Integral The definite integral ∫ a b f ⁡ ( x ) ⁢ ⅆ x of a function f ⁡ ( x ) defined on the interval [ a , b ] is given by ∫ a b f ⁡ ( x ) ⁢ ⅆ x = lim ‖ P ‖ → 0 ∑ i = 1 n f ⁡ ( x ¯ i ) ⁢ Δ x i where x ¯ i is a point in the i th subinterval of the partition P = { a = x 0 < x 1 < x 2 < ⋯ < x n = b } of the interval [ a , b ] , Δ x i = x i − x i − 1 , and ‖ P ‖ = max { Δ x i } . The sum ∑ i = 1 n f ⁡ ( x ¯ i ) ⁢ Δ x i is called a Riemann sum. The function f is integrable on [ a , b ] if the preceding limit exists. If f is integrable on [ a , b ] , then ∫ a b f ⁡ ( x ) ⁢ ⅆ x = lim n → ∞ b − a n ⁢ ∑ i = 1 n f ⁡ ( a + i ⁢ b − a n ) In particular, if f is continuous on [ a , b ] , then f is integrable on [ a , b ] . For positive-valued functions f , the sum b − a n ⁢ ∑ i = 1 n f ⁡ ( a + i ⁢ b − a n ) can be interpreted as the sum of areas of rectangles of base b − a n with height determined by the value of the function f at right endpoints of subintervals. For example, assume a = − 1 , b = 1 , n = 10 , and f ⁡ ( x ) = 1 x 2 + 1 . Caution
Use special care when working with improper integrals and make certain that answers look reasonable. Limits that straddle a discontinuity, such as ∫ − 1 1 1 x 2 ⁢ ⅆ x or ∫ − π π 1 + sin ⁡ x ( x − cos ⁡ x ) 2 ⁢ ⅆ x , should be avoided entirely. The latter will take a very long time to return a result and then simply returns the form you entered. Any time the system appears to hang up like this, examine the expression for a discontinuity.
