Saving Documents in Other Formats and Location At times you may wish to share Version 6 files with others who do not have the software. The easiest way to do this is to export the file and save it as a .tex file. You can also export your work in html format for use in Web publications. To export a document as a LATEX file documents!export as LaTeX@export as \LaTeX Open the document and choose File Export TEX... The Export TEX File dialog box appears.
