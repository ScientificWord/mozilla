Indefinite Integration

An antiderivative of a function f ⁡ ( x ) is any function g ⁡ ( x ) whose derivative is f ⁡ ( x ) . If g ⁡ ( x ) is an antiderivative of f ⁡ ( x ) , then g ⁡ ( x ) + C is another antiderivative. In fact, every antiderivative is of the form g ⁡ ( x ) + C for some constant C .

Entering and Evaluating Indefinite Integrals The indefinite integral of f ⁡ ( x ) is the family of all antiderivatives of f ⁡ ( x ) and is denoted ∫ f ⁡ ( x ) ⁢ ⅆ x . To enter an indefinite integral Choose Insert > Math Objects > Operator and choose ∫ . Type the expression to be integrated. Type ⅆ x . It is common, although hardly necessary, to add a thin space between f ⁡ ( x ) and ⅆ x in an integral ∫ f ⁡ ( x ) ⁢ ⅆ x . The Thin Space (found under Insert > Spacing Objects > Horizontal Spaces) is for readability only, and in no way affects the way in which an integral is interpreted by the underlying computing engine.
 To evaluate an indefinite integral Place the insert point anywhere in an integral and choose Evaluate.
