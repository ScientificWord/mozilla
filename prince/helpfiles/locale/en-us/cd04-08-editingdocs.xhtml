Information for Power Users Hints and tipsHints and Tips navigation!quickarrow keys Use the arrow keys for quick navigation and editing. To search forward for a specific character, hold down the right arrow key and type the character. To search backward for a specific character, hold down the left arrow key and type the character. To search forward for a group of characters, hold down the down arrow key and type the characters. To search backward for a group of characters, hold down the up arrow key and type the characters. Because of this functionality, holding down any of the arrow keys will not repeatedly move the insertion point. To do so, press the arrow key twice , holding it down the second time. Edit items with the context menu . For instance, to change a table, right-click the table.
