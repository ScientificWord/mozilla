Information for Power Users Hints and tipsHints and Tips Use the arrow keys for quick navigation and editing. To search forward for a specific character, hold down the right arrow key and type the character. To search backward for a specific character, hold down the left arrow key and type the character. To search forward for a group of characters, hold down the down arrow key and type the characters. To search backward for a group of characters, hold down the up arrow key and type the characters. Because of this functionality, holding down any of the arrow keys will not repeatedly move the insertion point. To do so, press the arrow key twice, holding it down the second time. Edit items with the context menu. For instance, to change a table, right-click the table. The context menu that appears has commands for adding and deleting rows and columns, among other things. Only images and tables that are using floating placement can be cross-referenced. On the Macintosh Macintosh Users Note using the function keys shortcuts typically defaults to requiring pressing the key combination fn and the desired function key. This can be changed to use the function keys as standard function keys by opening System Preferences and using the Keyboard shortcut and then turning on the appropriate check box.
