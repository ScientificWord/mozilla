Creating Animated 2D Plots

Animated Implicit Plots To make an animated implicit plot Type an equation in three variables. With the insert point in the expression, choose Compute > Plot 2D Animated > Implicit. The following animation shows the effect of the parameter t on the rectangular equation x 2 + t ⁢ y 2 = 1 as t varies from − 1 (which yields a hyperbola) to + 1 (which yields the unit circle). For this animation, select the frame and open the Graph User Settings dialog box. On the Items Plotted page, choose Edit Plot, and set − 2 ≤ x ≤ 2 , − 2 ≤ y ≤ 2 , and − 1 ≤ t ≤ 1 . Click the View tab and select Equal Scaling Along Each Axis. (You may notice a substantial delay as an animated implicit plot is generated.)



 Compute > Plot 2D Animated > Implicit x 2 + t ⁢ y 2 = 1 When you click this plot, the plotting tools become visible near the top of the screen. Use the slider to control the animation or click the triangle to view an animation. [ Return to main section ]
