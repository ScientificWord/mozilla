For example, x ⁢ y = 1 implicitly determines y as a function of x . This example is easily solved to give the explicit formula y = 1 / x . Many other equations cannot easily be solved for one of the variables. Also, some equations, such as x 2 + y 2 = 1 , do not determine a function, but pieces of the curves determined by such equations are functions. Implicit Differentiation, an item on the Calculus submenu, finds derivatives from an equation without explicitly solving the equation for any one variable. It is important to remember this variable in order to interpret the result, because the derivative is returned in the prime notation y ′ . However, it provides the basis for Newton's method , which is usually much faster than direct iteration. Newton's method is based on the observation that the tangent line is a good local approximation to the graph of a function. Let ( x 0 , f ⁡ ( x 0 ) ) be a point on the graph of the function f . The tangent line is given by the equation y − f ⁡ ( x 0 ) = f ′ ⁡ ( x 0 ) ⁢ ( x − x 0 ) This line crosses the x -axis when y = 0 . The corresponding value of x is given by x = x 0 − f ⁡ ( x 0 ) f ′ ⁡ ( x 0 ) In general, given an approximation x n to a zero of a function f ⁡ ( x ) , the tangent line at the point ( x n , f ⁡ ( x n ) ) crosses the x -axis at the point ( x n + 1 , 0 ) where x n + 1 = x n − f ⁡ ( x n ) f ′ ⁡ ( x n ) The Newton iteration function for a function f is the function g defined by g ⁡ ( x ) = x − f ⁡ ( x ) f ′ ⁡ ( x ) Given a first approximation x 0 , Newton's method produces a list x 1 , x 2 , … , x n of approximations to a zero of f .
