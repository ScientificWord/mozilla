Solving Polynomial Equations

There are four options on the Solve submenu: Exact, Integer, Numeric, and Recursion. The option Exact is general in nature and is used in most situations. It returns symbolic solutions when it can and numerical solutions otherwise. If any of the components of the problem use numerical notation, the response is a numerical solution. The three options Integer, Numeric, and Recursion are used in more specialized situations.

From the View menu, check Helper Lines or Input Boxes to help place equations in a matrix or display. To create a system of equations using a matrix Choose Insert > Math Objects > Matrix. Set the number of rows equal to the number of equations. Set the number of columns to 1 , and choose OK. Type the equations in the matrix, one equation to a row. To create a system of equations using a display Choose Insert > Math Objects > Display. Type the equations in the display, one equation to a row, adding rows as needed with the Enter key. To solve a system of equations Create a system of equations and leave the insert point in the system. Choose Compute > Solve > Exact. If a dialog box opens asking Variable(s) to Solve for, type the variable name(s) in the box, separated by commas. Following are examples for systems of two equations.
