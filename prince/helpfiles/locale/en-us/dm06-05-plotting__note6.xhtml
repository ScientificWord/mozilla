Creating Animated 2D Plots

Animated Parametric Plots in Polar Coordinates To make a parameterized animated plot in polar coordinates Type an expression of the form ( r ⁡ ( s , t ) , θ ⁡ ( s , t ) ) , and choose Compute > Plot 2D Animated > Polar. The following animation shows the effect of the parameter t in the polar equation r = 1 − t ⁢ cos ⁡ θ as t varies from − 2 to 2 . The limaçon, also called the limaçon of Pascal, is a polar curve of the form r = b + a ⁢ cos ⁡ θ . For the following animation, select the frame and open the Graph User Settings dialog box. Click the Items Plotted tab, choose Edit Plot, and set − 3.14159 ≤ θ ≤ 3.14159 and − 2 ≤ t ≤ 2 . On the View page, select Equal Scaling Along Each Axis.



 Compute > Plot 2D Animated > Polar ( 1 − t ⁢ cos ⁡ θ , θ ) When you click this plot, the plotting tools become visible near the top of the screen. Use the slider to control the animation or click the triangle to view an animation. [ Return to main section ]
