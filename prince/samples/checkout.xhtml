Integrating Term by Term

Theorem

Consider the power series f ⁡ ( z ) = ∑ n = 0 ∞ a n ⁢ z n ,  | z | < R ⁣ ( R ≠ 0 )

Let C be a simple piecewise smooth curve which lies inside the circle of convergence. Then we can integrate the power series term by term: ∫ C ( ∑ n = 0 ∞ a n ⁢ z n ) ⁢ ⅆ z = ∑ n = 0 ∞ a n ⁢ ∫ C z n ⁢ ⅆ z

Proof

The function f ⁡ ( z ) defined by the power series is continuous on C , so the integrals in (one) are well-defined. We need to show that

lim n → ∞ | ∫ C [ f ⁡ ( z ) − ∑ k = 0 n a k ⁢ z k ] ⁢ ⅆ z | = 0

Since C lies inside the circle of convergence, the series converges uniformly on C to f ⁡ ( z ) . For any ε , there is an N ⁡ ( ε ) so that, for all z on C , n ≥ N ⁡ ( ε ) ⇒ | f ⁡ ( z ) − ∑ k = 0 n a k ⁢ z k | < ε

By the triangle inequality for integrals and the above inequalities, for n ≥ N , | ∫ C [ f ⁡ ( z ) − ∑ k = 0 n a k ⁢ z k ] ⁢ ⅆ z | ≤ ε ⋅ ( length of C )

Since ε is arbitrary, the limit in (two) is zero.
